MINI MANUAL:

  • Para referenciar IDs --> #id {}
  • Para referenciar clases --> .class {}
  • Elementos concretos dentro de elementos --> tag#id {}
  • Seleccionar varios elementos --> body, div, p {}

En los archivos.css sólo se permiten comentarios en bloque " /* COMENTARIO */ "

UNIDADES RELATIVAS:

px:
Tamaño relativo respecto a la resolución de la pantalla.
%:
Porcentaje relativo al elemento actual en el cual estamos aplicando el estilo.
em:
Tamaño relativo de la fuente que tiene establecido el navegador.
rem:
El tamaño de la fuente del elemento raíz.
vh:
1% Altura del dispositivo de visionado (variable dinámicamente).
vw:
1% Anchura del dispositivo de visionado (variable dinámicamente).
vmin (vm):
El menor de los valores, 1% altura o anchura, del dispositivo de visionado (variable dinámicamente).
vmax:
El mayor de los valores, 1% altura o anchura, del dispositivo de visionado (variable dinámicamente).
Diferencia entre EM y REM:

HTML = 1em (16px)
Body = 1.3em (16px x 1.3 = 20.8px)
Div = 1.3em (20.8px x 1.3 = 27.04px)

La unidad "em" es relativa a la clase padre, por lo tanto, escalable.

SELECTORES:

Ejemplo de selectores combinados:

elem1 elem2 { estilos }

Con div#selectoresCombinados p { background-color: darkviolet; } se aplican los estilos al p dentro del div con id="selectoresCombinados"

Ejemplo de selectores con atributos:

elem1[attr="value"] { estilos }

Mediante el selector de atributos, se aplican los estilos al elem1 que contenga el attr con el valor "value".
En este caso se aplica background : "skyblue" al elemento p con la class="pSelectorAttr".

Ejemplo de selector de hijos:

elem1 > elem2 { estilos }

Mediante el selector de hijos, se aplican los estilos al elem2 perteneciente al elem1.
En este caso se aplica background : "forestgreen" al elemento p perteneciente a div#selectoresHijos.

Ejemplo de selector de adyacentes:

elem1 + elem2 { estilos }

Mediante el selector de adyacentes, se aplican los estilos a elem2 inmediatamente posterior al elem1.
En este caso se aplica background : "orangered" al elemento p inmediatamente posterior al elemento strong.
